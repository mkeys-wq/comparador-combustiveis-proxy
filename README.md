# Combustível+ — preços reais de combustível

App para comparar preços de combustível em Portugal, com dados reais e
atualizados diariamente da DGEG/ERSE. Mesma base visual/técnica do CARGA+
(mapa a ecrã inteiro, satélite, rota real, favoritos, instalável).

## Correr localmente
```bash
npm install
npm start
```
Depois abre http://localhost:3000

## Publicar no Render
1. Sobe esta pasta para um repositório no GitHub.
2. Em render.com: New + → Web Service → liga o repositório.
3. Build Command: `npm install`  ·  Start Command: `npm start`  ·  Plano: Free.
